# DETAILX Chicago

A fresh, premium static website for DETAILX Chicago, a mobile detailing business serving Chicago drivers.

## What is included

- Responsive homepage with luxury automotive positioning
- Services for Interior Detailing, Exterior Detailing, Ceramic Coating, Window Tint, Polishing, and Paint Correction
- Gallery, about, process, why choose us, reviews, FAQ, contact, and footer sections
- Front-end booking flow with service, date, time, contact, vehicle, location, and notes fields
- Calendar-style date picker and polished time slot selection
- Dependency-free HTML, CSS, and JavaScript for easy hosting

## Project structure

```text
.

```

## Run locally

Open `index.html` directly in a browser.

If you want a local server later, install a simple static server or open the folder in any editor that includes one. The project has no required build step.

## Future backend integration

The booking form is front-end only for now. The submit handler in `scripts/app.js` is the right place to connect a CRM, email service, scheduling API, or custom backend endpoint.
